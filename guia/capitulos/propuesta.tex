\subsection*{Tecnología}
Para el desarrollo del proyecto se usarán las heramientas iwlist para la adquisición de datos y para la \gui las librerías que el estudiante crea conveniente.



Este proyecto ha sido realizado mediante herramientas libres, lo cual reduce drásticamente los costes de licencias de software privado. \\

El lenguaje de programación utilizado en el desarrollo es C++ y se han utilizado las librerías gráficas Qt para el interfaz.
 Para el entendimiento de éstas librerías, se han utilizado principalmente dos libros de desarrollo en el framework QtCreator\footnote{http://qt.nokia.com/products/developer-tools/} y foros en internet.\\

El hecho de utilizar herramientas gratuitas, ha simplificado varios problemas; por un lado, al crear informes en formato PDF, la idea más sencilla ha sido utilizar \LaTeX , lenguaje con el que se está familiarizado y, por otro lado, el uso de GNUPlot ha simplificado la creación de gráficos sin tener que pasar a utilizar OpenGL u otras librerías para generación de gráficos.\\
